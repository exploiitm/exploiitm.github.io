_Disclaimer_: This is in no way a complete reference for all the security-related resources. If you are looking for something like that, have a look at [sbilly/awesome-security](https://github.com/sbilly/awesome-security). The resources listed below are just the resources which the Cybersecurity Club of IIT Madras uses to train new joiners.

# General resources

 - [OverTheWire](https://overthewire.org/wargames/): Really fun wargames to teach you everything from bash to web security.
 - [Secure Systems Engineering](https://sites.google.com/cse.iitm.ac.in/cs6570-2021/home): Course by IIT Madras about exploitation and security in modern systems.
 - [Baeldung](https://www.baeldung.com/cs/category/security): Baeldung blogposts on security.
 - [CyberChef](https://gchq.github.io/CyberChef/): Swiss army knife, built by the GCHQ, for solving commonly occurring tasks in CTF questions.
 - [HackTricks](https://book.hacktricks.xyz/welcome/readme): Collection of tons of attacks and exploitation techniques, with explanation.
 - [d00mfist's notes](https://d00mfist.gitbooks.io/ctf/content/): Comprehensive notes based on CTF experience.
 - [Team bi0s wiki](https://wiki.bi0s.in/): Introductory resources classified on the basis of the various types of CTF challenges.
 - [Information Security Stack Exchange](https://security.stackexchange.com/): Collection of high-quality questions and answers related to security.

# Binary Exploitation & Reverse Engineering

 - [Ret to Rop](https://ret2rop.blogspot.com/): Fun blog for understanding the fundamentals of common binary exploitation techniques.
 - [Nightmare](https://guyinatuxedo.github.io/): Book/course on binary exploitation and reversing techniques, which uses CTF questions to explain the various exploits in a practical manner.
 - [Modern Binary Exploitation (CSCI 4968)](https://web.archive.org/web/20210710080726/http://security.cs.rpi.edu/courses/binexp-spring2015/): Course by Rensselaer Polytechnic Institute on modern binary exploitation techniques.

# Web Security

 - [Web Security (CS253)](https://web.stanford.edu/class/cs253/): Stanford's course on web security.

# Cryptography

 - [cryptopals](https://cryptopals.com/): Crypto challenges meant to teach and demonstrate attacks on real-world crypto.
 - [Cryptography Stack Exchange](https://crypto.stackexchange.com/): Collection of high-quality questions and answers related to cryptography.

# Challenges

 - [picoCTF](https://picoctf.org/): CTF competition for beginners, organized by CMU.
 - [HackTheBox](https://www.hackthebox.com/): Great platform to practice cybersecurity by solving challenges and hacking boxes.
 - [TryHackMe](https://tryhackme.com/): Interactive, practical real-world cybersecurity lessons.
 - [crackmes.one](https://crackmes.one): A collection of binaries which can be cracked, great for improving reversing and binary exploitation skills.
 - [VulnHub](https://www.vulnhub.com/): Vulnerable machines which can be hacked for practice and fun.
